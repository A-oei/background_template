## background_template 项目

项目模仿[element-ui](https://element.eleme.cn/#/zh-CN/component/installation)实现了一些常用的插件，样式基本参考了`element`框架，实现了部分常用的功能

* `input`输入框
* `Message`消息提示
* `sidebar`侧边导航栏
* `Table`表格展示
* `Dialog`弹窗
* `Button`按钮
* `Card`展示面板
* `DatePicker`日期选择器
* `Dropdown`下拉菜单
* `Select`选择器
* `Pagination`分页器
